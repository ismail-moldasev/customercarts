# Livestock Cards (MVP макет)

Минимальный одностраничный прототип маркетплейса для быстрого создания и поиска карточек по живому скоту, мясу и молочной продукции.

## Как запустить
1. Откройте `index.html` в браузере или поднимите локальный сервер: `python -m http.server 8000` и перейдите на `http://localhost:8000`.
2. Все данные (черновики, избранное, история) сохраняются в `localStorage`, так что прототип работает офлайн.

## Что внутри
- **Главный экран** с тремя крупными кнопками: Продавец, Клиент, Сохранённые/История.
- **Продавец → Живой скот**: умная форма с таблицей правил, динамическими полями и предпросмотром карточки.
- **Продавец → Мясо / Молоко**: отдельные упрощённые формы с предпросмотром.
- **Клиент**: витрина с поиском/чипами-фильтрами и карточками для трёх направлений.
- **Сохранённые/История**: вкладки «Избранное», «Мои карточки», «История просмотров».

Дизайн — минимализм, крупные кнопки и мягкий градиентный фон в духе «спокойного пульта» рынка.
